## Installation

Get the latest version and run the following commands:
```
$ git clone https://github.com/joseluisluri/nand.me-viewer.git
$ cd nand.me-viewer
$ npm install -g
$ tsc -p src
```

For development, you can use the integrated lite-server:

```
$ npm start
```
